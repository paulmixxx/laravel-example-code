# Требования
- php: >=5.6
- [composer](https://getcomposer.org/)
- make

<br>

# Установка
```bash
make install
```

# Использование

##### Проверить код
```bash
make check
```
##### Исправить код
```bash
make fix
```

##### Запуск тестов
```
make test
```
